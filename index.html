Tag H

Esta tag é responsavel por diferenciar os titulos, tamanhos, em nivel de hierarquia.
Começando pelo H1, Que é o maior titulo, e assim adiante os tamanhos irão diminuindo conforme o numero for aumentando H2, um titulo menor e assim sucessivamente até o H6, que é o ultimo.

Tag P

A tag P é responsavel por identificar um paragrafo, comecar novos paragrafos

Tag HR

Tag de uso simples, para dar um melhor acabamento, pode ser usada em qualquer parte do corpo, ela é responsavel por fazer um traco no meio da tela.

LISTAS

Listas são uma das tags mais importantes do html, com ela dá pra fazer alem de menu, colocar uma ordem de conteudos, agrupar elementos numa lista só. Poderá ainda ordenar, em listas ordenadas númeradas, e listas não ordenadas, que sera representada por um pontinho, ou, com ajuda do css poderá mudar o simbolo.

TITLE

Tag que mudará apenas a abá no navegador, usada no head, tag de uso simples.

LINKS

Link: Funciona como se fossem ligaçoes, endereços ou buscas, por exemplo: Liga um link que foi pegado de alugum outro lugar... Tag muito importante.
